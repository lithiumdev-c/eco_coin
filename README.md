# 🌱 EcoCoin MVP

EcoCoin is a blockchain-based loyalty system designed to reward environmentally conscious actions. Using Computer Vision, the platform verifies recycling efforts and rewards users with digital tokens.

   ```cd backend
     pip install -r requirements.txt

      ```python -m venv .venv
